# Trevo — Sistema SaaS de Agendamento

> Sistema em uso real com clientes ativos

Sistema de agendamento multi-tenant desenvolvido para atender diferentes tipos de negócio, como barbearias, clínicas e prestadores de serviço.

O sistema já está em uso por clientes reais.

## Principais funcionalidades

- Agendamento de horários com controle de disponibilidade  
- Prevenção de conflitos de horários  
- Painel administrativo para gestão de serviços e clientes  
- Estrutura multi-tenant (suporte a múltiplos usuários/negócios)  
- Sistema preparado para evolução para modelo SaaS

---

## Estrutura técnica

O sistema foi desenvolvido com foco em backend e organização de regras de negócio:

- PHP para processamento das regras do sistema  
- MySQL para modelagem e persistência de dados  
- JavaScript para interações e validações no frontend  

A arquitetura foi planejada para permitir evolução contínua e adaptação para modelo SaaS escalável.

---

## Regras de negócio implementadas

- Controle de horários disponíveis  
- Validação de dados no agendamento  
- Prevenção de conflitos de horários  
- Organização de dados por usuário (multi-tenant)
